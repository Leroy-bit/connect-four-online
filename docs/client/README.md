#### Building and running the client
!> **You must be in the `client` directory**

**Requirements:**
- Node js - 18.18.0 and higher

**Install dependencies**

```bash
npm install
```

***

##### Build for production

```bash
npm run build
```

##### Run development server

```bash
npm run dev
```
